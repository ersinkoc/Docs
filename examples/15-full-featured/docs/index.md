# Full-Featured Documentation

This example demonstrates all the features of @oxog/docs.

## Features Included

- **Zero-config**: Works out of the box
- **Custom configuration**: Fine-tuned settings
- **Theme customization**: Branded styling
- **Plugin system**: Extend functionality
- **Multiple frameworks**: React, Vue, Svelte, Solid, Vanilla
- **File-based routing**: Automatic page discovery
- **Markdown support**: Rich content authoring
- **Syntax highlighting**: Beautiful code blocks
- **Search**: Find content quickly
- **Responsive design**: Works on all devices
- **Dark mode**: Easy on the eyes

## Getting Started

```bash
npx @oxog/docs init
cd my-docs
npm install
npm run dev
```
